**__All user access__**
**hello** – Say hello to me!
**calc** – Sends a link to any currently relevant damage calculators
**chart** – Sends an image of the current type weakness chart
**invite** – Sends a perminant invite link to this server
**rule** – Sends one of the four main rules for this server
**weak** – Sends all the type advantages for a type or type combination

**__Moderator only commands__**
**delete** – Deletes a specified number of messages
**verify** – Changes the verification level of the server

Remember to start messages with `!` if you're trying to use a command!
